2. 我能幫老師做哪些事？

出考卷與測驗

依照單元、年級、難易度產生選擇題、填充題、問答題與答案解析。

例：幫五年級自然「植物構造」出 20 題，其中 5 題要有生活情境。

改寫教材

把艱深內容改成學生聽得懂的版本，或整理成重點表格。

例：把課本段落改成三分鐘口頭講解稿，再附一張重點心智圖大綱。

製作學習單

設計引導問題、活動步驟、檢核表與延伸任務。

例：做一份國語課「人物特質分析」學習單，含例句與學生自評。

班級經營

協助設計班規、獎勵制度、座位討論原則與衝突處理話術。

例：幫我寫一套適合中年級的分組合作規則，語氣要溫和但明確。

家長聯絡

把老師想表達的事情改寫成清楚、禮貌、避免誤會的訊息。

例：寫一則通知家長孩子最近作業常缺交的訊息，不責備但要有提醒力。

差異化教學

同一份內容可以改成基礎版、普通版、挑戰版，照顧不同程度學生。

例：把數學分數應用題改成三種難度，每種各 5 題。

活動與課程設計

幫忙規劃課堂流程、分組任務、討論題目、時間分配與收尾回饋。

例：設計一堂 40 分鐘的防災教育課，包含暖身、任務、分享與評量。

簡報與網頁素材

整理投影片架構、撰寫標題、製作簡單 HTML 展示頁或課堂小工具。

例：做一個成語配對小遊戲網頁，深色背景、按鈕要大，適合投影操作。

行政文字與會議資料

協助整理會議紀錄、活動企劃、公告、回饋表與成果報告。

例：把校外教學流程整理成家長通知單與教師工作分配表。

3. 老師最該裝的 Antigravity 插件 / 擴充功能

如果 Antigravity 使用 VS Code 相容的擴充功能，以下這些最適合教學現場，因為它們能讓「做教材、改網頁、整理文字、檢查成果」更順手。

Live Server 即時預覽 HTML 教材、小遊戲與班級網頁。老師修改檔案後重新整理瀏覽器，就能立刻看到成果。
Prettier 自動排版 HTML、CSS、JavaScript，讓學生作品或老師教材的程式碼更整齊，方便教學示範。
Markdown All in One 適合寫課程講義、閱讀筆記、班級公告草稿，能快速產生標題、清單與目錄。
Code Spell Checker 檢查英文單字與常見拼字錯誤，製作雙語教材、英文學習單或網頁按鈕文字時很實用。
GitLens 如果課堂作品會一版一版修改，它可以協助追蹤變更，知道哪次改了什麼，也比較容易回到前一版。

4. 怎麼跟我說話，才會得到最好的回答？

Do：這樣說更有效

  • 說清楚對象：年級、科目、學生程度、使用情境。
  • 給我格式：要表格、講稿、考卷、家長通知、HTML 網頁，都可以直接說。
  • 給我限制：字數、時間、題數、難度、語氣、是否要附答案。
  • 先給草稿也沒關係：我可以幫你修得更清楚、更友善、更有條理。
  • 要求我檢查：請我找錯字、檢查題目難度、確認網頁能不能打開。

Don't：避免這樣問

  • 不要只說「幫我做教材」，最好補上科目、單元與學生年級。
  • 不要一次塞太多不同任務，可以先完成一個版本，再請我調整。
  • 不要假設我知道班上的特殊狀況，重要背景請直接告訴我。
  • 不要把 AI 產出直接當最終答案，考題、通知與評量仍建議老師最後確認。
  • 不要害怕改口，你可以說「再簡單一點」「更正式」「改成表格」。
